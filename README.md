### A Full-Stack Task Management Application built with React (Frontend) + Node.js, Express, MongoDB (Backend) using JWT Authentication and Role-Based Access Control (Admin / User).

## Authentication
- User Registration & Login
- JWT-based Authentication
- Secure password hashing using (bcryptjs)

## Roles
### Admin
- View all tasks

### User 
- View only their own tasks

## Tasks 
- Create Task 
- Get Task (roles based)
- Update Task (owner only)
- Delete Task (owner only)

## Frontend 
- React.js 
- Axios
- Context API 
- Tailwincss 

## Backend 
- Node.js
- Express.js
- MongoDB + Mongoose 
- JWT (jsonwebtoken)
- bcryptjs (hashing password)
- dotenv

## Backend setup 
- cd backend 
- npm install

### start backend server 
npm run dev or node app.js

backend on port - http://localhost:5000


## Frontend Setup 
cd frontend 
npm install 
npm run dev

frontend on port http://localhost:5173
